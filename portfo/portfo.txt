Project Title: "EDA_GPT: AI-Powered Exploratory Data Analysis"
Project Link: "https://github.com/Tajaar/EDA_GPT"
Domains: ["AI", "Data Analysis", "NLP"]
Description: "EDA_GPT is an open-source tool that automates exploratory data analysis using natural language interfaces. It supports both structured data (CSV, XLSX, SQL) and unstructured data (PDFs, images), enabling users to generate visualizations, perform data cleaning, and derive insights without extensive coding."
